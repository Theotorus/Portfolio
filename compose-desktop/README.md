Cette version du portfolio est développée avec Jetpack Compose for Desktop (Kotlin).
Elle fait partie du projet multi-framework destiné à démontrer mes compétences en conception d’interfaces et en développement multiplateforme.

# Objectif
Ce projet met en avant ma capacité à créer une application desktop moderne avec Compose, en respectant une architecture claire et une expérience utilisateur cohérente avec les autres versions (React, Angular…).

# Technologies utilisées
Kotlin

Jetpack Compose for Desktop

Gradle (build et packaging)

MVVM architecture (via MainViewModel)

Material Design (Compose UI components)

Les exécutables (Windows uniquement pour le moment) sont disponibles via les GitHub Releases du dépôt principal.
